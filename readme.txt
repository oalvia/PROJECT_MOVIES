PROYECTO MOVIES
OSCAR ALVIAREZ



SE CUMPLEN LOS OBJETIVOS DE LAS PRACTICAS Servidor Movies I Y Servidor Movies II.

Puntos realizados:

Crear un servidor de películas.
El servidor debe utilizar Express.js
Base de datos mediante MongoAtlas, utilizando mongoose.
Semilla con el nombre movies.seed.js que cargue en la base de datos una selección de películas.
Crear un endpoint get que devuelva todas las películas.
Crear un endpoint get que devuelva una película según su _id
Crear un endpoint get que devuelva un valor por su titulo.
Crear un endpoint get que devuelva los documentos según su género.
Crear un endpoint get que devuelva las películas que se han estrenado a partir de un año a seleccionar.
Completar un CRUD completo para las peliculas (movies)
Base de datos que contenga la información de los cines (cinemas)
Crear un directorio de rutas y mover las rutas ya creadas a movie.routes.js
Crear un método post de Movies para crear una nueva película.
Crear un método put de Movies para modificar una película.
Crear un método delete de Movies para eliminar una película.
Crear un modelo Cinema el cual tendrá las propiedades: 
    name (campo requerido).
    location (campo requerido).
    movies [ ] (relación con la colección movies).
Crear los métodos GET, POST, PUT y DELETE de la colección Cinema.
Insertar dos documentos en la colección cinema que tengan al menos una relación con una película.
Controlar los errores.

Adicionales:

Control de seciones con cookies.
Autenticación.
Subir archivos de imagenes para las peliculas.
Se modifica el Endpoint de actualización de peliculas para poder insertar las imagenes.
